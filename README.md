# Health Insurance Premium Prediction Using Regression Models

## Project Overview

Health insurance premiums are calculated based on several health risk factors such as age, BMI, smoking habits, gender, number of children, and region.

In this project, I used the Medical Cost Personal Dataset from Kaggle and built multiple regression models to predict health insurance premiums. The project includes data preprocessing, exploratory data analysis (EDA), feature engineering, model building, and model evaluation.

## Models Used

- Linear Regression
- Lasso Regression
- Ridge Regression
- Decision Tree Regressor

## Evaluation Metrics

- R² Score
- RMSE
- MAE
- MSE

## Overall Observation

After comparing all the regression models, the Decision Tree Regressor achieved the best overall performance. It predicted health insurance premiums more accurately than the other models based on the evaluation metrics.
